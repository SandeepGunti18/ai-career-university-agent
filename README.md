# AI Career & University Decision Agent

This project is an AI agent that helps students choose the best academic path and universities based on:

- interests
- preferred course
- budget
- domestic/international preference

## Features

- Uses modular tool architecture
- Simulates LPI tool workflow
- Input validation included
- Error handling with try/catch
- Explainable recommendations
- Reasoning output included

## LPI Tools Used

### 1. query_knowledge
Used to evaluate curriculum quality, placements, rankings, and ROI.

### 2. get_insights
Used to personalize recommendations based on budget and location preference.

## Tech Stack

- JavaScript
- Node.js
